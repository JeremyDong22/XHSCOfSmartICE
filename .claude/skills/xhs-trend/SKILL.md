# 小红书市场趋势分析

## 概述

利用项目已有的小红书爬虫能力，分析餐饮AI相关的热门内容趋势、竞品动态、热门标签，为内容策略提供数据支撑。

## 参数

`/xhs-trend [关键词或方向]`
- 无参数：分析餐饮AI整体趋势
- 指定关键词：分析特定话题的热度
- 示例：`/xhs-trend 餐饮AI`
- 示例：`/xhs-trend 智能餐厅`

## 执行流程

### 1. 确定分析范围

根据用户需求确定要分析的维度：
- **话题热度**：特定关键词的帖子数量和互动趋势
- **竞品动态**：同类账号在发什么内容
- **热门标签**：哪些标签的搜索量在涨
- **内容风格**：当前高赞帖子的标题/封面/正文风格

### 2. 数据采集

使用两种方式采集数据：

**方式A：利用项目爬虫（推荐）**
通过后端 API 调用已有的小红书爬虫：
- `POST /api/scrape/start` — 搜索关键词，抓取帖子列表
- 分析抓取结果中的高赞帖子（按点赞/收藏排序）
- 提取标题模式、标签使用、内容结构

**方式B：DevTools MCP 手动探索**
当需要更深入的分析时，通过浏览器直接探索：
- navigate_page → 小红书搜索页
- 搜索目标关键词
- take_snapshot → 分析搜索结果页的帖子列表
- 点击高赞帖子查看详细内容和评论

### 3. 趋势分析

对采集到的数据进行分析：

**话题热度**：
- 搜索结果数量级（几百 vs 几千 vs 几万）
- 高赞帖子的发布时间分布（是否是近期热点）
- 互动量级（平均点赞/收藏/评论）

**内容模式**：
- 高赞帖子的标题公式（疑问？数字？情绪词？）
- 封面风格（文字配图 vs 实拍 vs 截图）
- 正文结构（故事型 vs 干货型 vs 对比型）
- CTA风格（评论引导 vs 私信引导 vs 关注引导）

**标签生态**：
- 高频标签有哪些
- 哪些标签出现在高赞帖子中
- 是否有新兴标签（之前没见过但最近频繁出现）

**竞品洞察**：
- 同类账号的发帖频率
- 他们的高赞内容是什么类型
- 他们的差异化定位是什么

### 4. 策略建议

基于分析结果，生成可执行的策略建议：

- **热点跟进**：哪些话题值得立即跟进发帖
- **标签调整**：建议更新标签策略
- **内容方向**：建议下一篇帖子的主题和类型
- **差异化**：我们的内容如何与竞品区分

### 5. 更新知识库

将有价值的发现更新到共享文件（位于 `.claude/plugins/xhs-automation/shared/`）：
- 新发现的高效标签 → 更新 `writing-formulas.md` 的标签策略
- 新的行业热点 → 更新 `brand-context.md` 的近期可用素材
- 竞品案例 → 更新 `brand-knowledge.md` 的大品牌AI案例

### 6. 输出报告

向用户展示趋势分析报告：
- 当前热门话题 Top 5
- 高赞内容模式总结
- 推荐的下一步行动（发什么、怎么发、用什么标签）
- 需要警惕的趋势（如某个话题已经过热、竞品在抢占某个定位）
